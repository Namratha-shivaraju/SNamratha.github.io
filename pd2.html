BitQueen - Creating Our Own Crypto-Tokens and Selling Them

We created our own crypto tokens called BitQueen and set up a token sale. Once the user logs in, they see the token sale homepage which shows the amount of time before the sale ends, a progress bar at the bottom that displays the amount of tokens that have been sold out, the token details, an option to buy the tokens and a button that redirects a user to their dashboard. The admin dashboard gives the admin’s personal details and allows them to make announcement. The admin can transfers a certain amount of tokens to the token sale to start the sale. A login page that allows already registered users to login to their accounts using their email id and password. A sign up page for users who aren’t registered to access the token sale home page.
