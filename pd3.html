Project information

  • Technologies: Python3, OpenCV2, OS, Sys, Numpy, Datetime, and Tkinter

Automatic Face Recognition Attendance Application

The project idea was born out of the simple need to help teachers take the class attendance in a quicker and easier fashion. Since the project works using a camera to send a live video feed to a computer - within which all the face-data is stored - any one present in the room would automatically be registered as “present” by the computer, thus eliminating any human interaction once set up.
